# Playwright UI Automation Framework

This is a Python-based UI automation framework using [Playwright](https://playwright.dev/python/) and [pytest](https://docs.pytest.org/en/stable/).

## Features

- Page Object Model (POM) design pattern for maintainability
- Supports Chrome browser automation
- Configurable pytest setup with fixtures
- Sample tests for homepage and login flows
- Ready for extension with additional test cases

## Prerequisites

- Python 3.12
- Google Chrome browser installed
- `playwright` Python package and dependencies
